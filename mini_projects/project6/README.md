# Project 6 — Ultrasonic Sensor Publisher

ESP32 reads an ultrasonic sensor and publishes distance values.  

- Topic: `/ultrasonic`  
- ROS2 Python node plots distance in a radar-like visualization

**Files:**  
- `esp32_ultrasonic_pub/` → ESP32 publisher C code  
- `ros2_ultrasonic_sub/` → Python subscriber  

**Goal:** Learn sensor integration and real-time MCU → ROS2 data flow.
