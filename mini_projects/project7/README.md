# Project 7 — Joystick Control

Simulated joystick using two buttons; publishes x and y values.

- Topic: `/joystick`  
- Python ROS2 node subscribes and moves a simulated robot

**Files:**  
- `esp32_joystick_pub/` → ESP32 publisher C code  
- `ros2_joystick_sub/` → Python subscriber  

**Goal:** Learn multi-dimensional input and ROS2 publishing.
