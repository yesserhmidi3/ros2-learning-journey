# Project 4 — Potentiometer Publisher

ESP32 reads an analog potentiometer and publishes the value.

- Topic: `/potentiometer`  
- ROS2 Python node plots values in real-time

**Files:**  
- `esp32_pot_pub/` → ESP32 publisher C code  
- `ros2_pot_sub/` → Python subscriber  

**Goal:** Learn analog input, data streaming, and plotting.
